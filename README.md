# Islamic Quotes Generator with AI Assistant

Une application web moderne qui affiche des citations islamiques inspirantes du Coran et des Hadiths authentiques, avec un assistant IA intégré.

## 🌟 Fonctionnalités

- 📖 **Citations aléatoires** du Coran et des Hadiths authentiques
- 🎯 **Filtrage par catégories** (Patience, Foi, Connaissance, Morale)
- 💾 **Système de favoris** avec LocalStorage
- 📋 **Copie des citations** en un clic
- 🤖 **Assistant IA** pour répondre aux questions sur l'Islam
- 🌙 **Mode sombre/clair**
- 📱 **Design responsive** (mobile, tablette, desktop)
- 🔄 **Animations fluides** avec Framer Motion
- 🌍 **Support multilingue** (français, anglais, arabe)

## 🚀 Technologies utilisées

- React 18
- Tailwind CSS
- Framer Motion
- React Router
- LocalStorage API
- OpenAI API (optionnel)
